## 搭建测试
> 本文档仅供参考，请勿在生产环境使用。

Fork本仓库，使用`Cloudflare Pages`连接到`GitHub存储库`

构建命令 `npm run docs:build`

构建输出目录 `docs/.vitepress/dist`

环境变量 变量名称`API_URL`值为你的`分享页API` 

分享页API参考: https://appleauto.pro/docs/api/share-api.html
